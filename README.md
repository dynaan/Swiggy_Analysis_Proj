# Swiggy Sales Analysis Dashboard

## Objective
Analyze Swiggy order data to identify sales trends, customer behavior, restaurant performance, and operational insights through an interactive Excel dashboard.

## Dataset
The dataset contains information related to:
- Orders
- Customers
- Restaurants
- Delivery Partners
- Cities
- Order Value
- Ratings

## Key Business Questions
1. Which restaurants generate the highest revenue?
2. Which cities contribute the most orders?
3. What are the peak ordering hours?
4. Which food categories are most popular?
5. What is the average order value across different cities?
6. Which restaurants have the highest customer ratings?
7. How does revenue vary month-over-month?
8. Which delivery partners complete the most orders?

## Dashboard Features
- Total Revenue KPI
- Total Orders KPI
- Average Order Value (AOV)
- Revenue by City
- Revenue by Restaurant
- Orders by Food Category
- Monthly Sales Trend
- Customer Rating Analysis
- Interactive Filters and Slicers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning Techniques

## Key Insights---
- Identified top-performing restaurants based on revenue.
- Determined peak ordering periods.
- Analyzed customer preferences across food categories.
- Compared city-wise sales performance.
- Evaluated restaurant ratings and customer satisfaction.

## Project Outcome
The dashboard enables business stakeholders to monitor sales performance, identify growth opportunities, and make data-driven decisions regarding restaurant partnerships and customer engagement.
